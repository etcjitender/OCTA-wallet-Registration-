## Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/octra-labs/wallet-gen.git
   cd wallet-gen
   ```

2. **Run the wallet generator webserver:**
   
   **Linux/macOS:**
   ```bash
   chmod +x ./start.sh
   ./start.sh
   ```
   
   **Windows:**
   ```bash
   start.bat
   ```
